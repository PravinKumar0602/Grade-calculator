# Student Grade Calculator 🎓

A simple, console-based Java application that calculates a student's total marks, average percentage, and assigns a grade based on performance.

## 🚀 Features
* Input student name and number of subjects.
* Calculate total marks and average percentage.
* Automatic grade assignment (A+, A, B, C, D, or Fail).
* Clean and easy-to-read console output.

## 🛠️ Requirements
* **Java JDK** (version 8 or higher)
* Any text editor or IDE (VS Code, IntelliJ, Eclipse)
